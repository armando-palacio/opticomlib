opticomlib |version|
====================

**OptiComLib** is a Python library for optical communication systems,
using coherent and direct detection modulation formats. It provides open 
source code for easy implementation of optical links. 
The project is hosted on `GitHub <https://github.com/armando-palacio/opticomlib.git>`_.


.. note:: 
   This project (and documentation) is under active development.
